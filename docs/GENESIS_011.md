# Genesis-011: Entry Gates and Signal Clarity

Genesis-011 adds tools that explain Edge15's entry decision instead of leaving the user to guess why a signal is READY, ENTER, WAIT, or risky.

## Added

- Entry Gate Checklist
- Late-entry warning
- Contradiction alert
- Do Not Chase warning
- Contract phase label
- Confidence Heat Strip
- Trade Quality Filter
- “Would Edge15 still enter now?” in Position Mode
- Trade Review unselect behavior
- Active review reset when clearing a position

## Notes

The decision engine itself remains anchored to the Genesis-008 settlement guardrails and Genesis-005 signal plan. Genesis-011 is primarily a transparency and mistake-prevention release.
