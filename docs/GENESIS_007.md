# Genesis-007 — Context Helpers + Persistent Trade Context

Genesis-007 improves clarity while preserving the working Genesis-006 AI Trading Desk foundation.

## Added

- Plain-English helper descriptions inside the main entry-mode metric boxes.
- Descriptions for Trade Plan, Entry Score, Opportunity, Trade Grade, Model Trust, and Signal Stability.
- Trade Plan context remains visible after pressing Entered OVER or Entered UNDER.
- Locked Position Mode now appears below the trade-context panel instead of replacing it.
- Position Mode keeps HOLD / CAUTION / DANGER while still showing the original trade setup context.

## Why this matters

Genesis-006 worked, but once a position was locked, several decision-context boxes disappeared. Genesis-007 fixes that by keeping the original trade context visible so you can compare what Edge15 liked at entry against how the trade is behaving after entry.
