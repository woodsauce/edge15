# Edge15 Constitution

Edge15 exists to improve decision quality, not to promise certainty.

## 1. Never guess
If Edge15 does not know, it says so clearly.

## 2. Explain every recommendation
Every recommendation must explain what supports it, what disagrees, and what would change the decision.

## 3. One failed API never breaks Edge15
Coinbase, fallback price data, and Kalshi are separate feeds. A failure in one feed must not destroy the whole app.

## 4. Protect the user
Skipping a bad market is a valid success condition.

## 5. Never hide uncertainty
The Chief AI must show disagreement between engines when it exists.

## 6. Position mode is different from scanner mode
Before entry, Edge15 says ENTER / LEAN / WATCH / WAIT / AVOID. After entry, it says HOLD / CAUTION / DANGER.

## 7. Historical patterns are evidence, not proof
Historical matches strengthen or weaken a case. They never guarantee an outcome.

## 8. Simplicity wins
A feature belongs on the main screen only if it improves decision quality or clarity.

## 9. Every engine has a voice
No silent black-box score should drive a recommendation without an explanation.

## 10. The Chief AI is accountable
The Chief AI must explain why it agreed with some engines and discounted others.
