# Genesis-017 — Focused Cockpit Cleanup

Genesis-017 removes clutter from the main Edge15 screen so the app is easier to use during live 15-minute markets.

## Main-screen focus

Visible by default:

- Current trade plan
- Countdown
- BTC price / reference / distance
- Settlement risk
- Performance Tracker
- Commitment Accuracy
- Entry Gate Checklist
- Late-entry / contradiction / do-not-chase warnings
- Position mode

Hidden from the main screen:

- Market Microstructure
- Market Story
- Confidence Heat Strip
- Why Not / Trade Plan
- AI Debate
- Trade Review + Learning Log

## Accuracy posture

Genesis-017 keeps the Genesis-015 protection behavior:

- Final 3 minutes blocked for fresh entries
- Payout-value gate
- Stricter ENTER behavior
- NO TRADE remains a protective outcome
